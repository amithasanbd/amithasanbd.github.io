- `data/gallery.json`

- `assets/profile.jpg`

## Deployment

Upload every file and folder from this package directly into the root of:

`amithasanbd.github.io`

Do not upload the ZIP file itself. Do not upload this package as a folder. The root must contain `index.html`.
